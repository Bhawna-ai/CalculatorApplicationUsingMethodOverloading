# CalculatorApplicationUsingMethodOverloading
This project is a simple Java-based calculator that demonstrates **method overloading**, **exception handling**, and **menu-driven user interaction**. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division using overloaded methods.

## Features
- Overloaded `add()` methods:
  - add(int, int)
  - add(double, double)
  - add(int, int, int)
- Subtraction: subtract(int, int)
- Multiplication: multiply(double, double)
- Division with error handling: divide(int, int)
- Menu-driven interface using Scanner
- Handles invalid input using try-catch

## How to Run
1. Clone the repository  
   `git clone https://github.com/your-username/CalculatorApp.git`
2. Compile the files  
   `javac Calculator.java UserInterface.java`
3. Run the program  
   `java UserInterface`
